大一
探索期

积极探索大学生活

分岔口

确定未来方向

科研读研
  • 保证绩点，为保研打基础

全力冲刺实习

  • 寒假过后开始刷题（LeetCode）
  • 背八股文（计算机网络、操作系统、数据库等）

  • 培养一项业余爱好（运动、音乐、旅行等）

学习建议

📚

基础为王

数据结构、算法、操作系统、计算机网络、数据库原理这些基础课程决定了你未来的上限。不要只追求热门技术栈。

💻

动手实践

理论学习很重要，但实践更重要。每学一门技术，都要做项目巩固。可以参与开源项目，积累实战经验。

🔄

持续学习

技术更新很快，要养成持续学习的习惯。关注技术博客、参加技术社区、阅读技术书籍。

🎯

明确方向

尽早确定自己的方向（前端/后端/移动端/算法等），有针对性地深入学习和实践。

🤝

团队协作

学会使用Git、参与团队项目、培养沟通能力。这些都是职场必备技能。

⏰

时间管理

平衡课程学习和项目实践。不要为了项目而忽视课程，也不要死读书而缺乏实践。

数据结构与算法

必修
学分: 5 大二上

计算机科学的核心课程，学习各种数据结构和算法设计与分析。

重点内容

  • 线性表、栈、队列
  • 树和二叉树
  • 图论基础
  • 查找和排序算法
  • 算法复杂度分析

代码示例

// 链表节点定义
struct ListNode {
    int val;
    struct ListNode *next;
};

// 链表反转
struct ListNode* reverseList(struct ListNode* head) {
    struct ListNode *prev = NULL;
    struct ListNode *curr = head;

    while (curr != NULL) {
        struct ListNode *next = curr->next;
        curr->next = prev;
        prev = curr;
        curr = next;
    }

    return prev;
}

// 快速排序
void quickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}

数据库原理

必修
学分: 4 大二下

学习关系型数据库的设计、SQL语言、事务处理、索引优化等。

重点内容

  • ER模型和数据库设计
  • 关系代数和SQL
  • 事务ACID特性
  • 索引和查询优化
  • 存储过程和触发器

代码示例

-- 创建表
CREATE TABLE students (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50) NOT NULL,
    email VARCHAR(100) UNIQUE,
    enrollment_date DATE DEFAULT CURRENT_DATE
);

-- 插入数据
INSERT INTO students (name, email)
VALUES ('张三', 'zhangsan@example.com');

-- 复杂查询
SELECT s.name, AVG(c.score) as avg_score
FROM students s
JOIN courses c ON s.id = c.student_id
WHERE c.semester = '2024-1'
GROUP BY s.id, s.name
HAVING AVG(c.score) >= 80
ORDER BY avg_score DESC;

-- 事务示例
START TRANSACTION;
UPDATE accounts SET balance = balance - 100 WHERE id = 1;
UPDATE accounts SET balance = balance + 100 WHERE id = 2;
COMMIT;

操作系统

必修
学分: 4 大三上

深入理解操作系统的核心原理：进程管理、内存管理、文件系统等。

重点内容

  • 进程与线程
  • CPU调度算法
  • 内存管理（虚拟内存、分页分段）
  • 文件系统
  • 死锁处理

代码示例

// 进程创建（fork）
#include <unistd.h>
#include <stdio.h>

int main() {
    pid_t pid = fork();

    if (pid == 0) {
        // 子进程
        printf("子进程: PID=%d\n", getpid());
    } else if (pid > 0) {
        // 父进程
        printf("父进程: PID=%d, 子进程PID=%d\n",
               getpid(), pid);
    } else {
        perror("fork失败");
    }

    return 0;
}

// 线程同步（互斥锁）
#include <pthread.h>

pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER;
int counter = 0;

void* increment(void* arg) {
    for (int i = 0; i < 100000; i++) {
        pthread_mutex_lock(&mutex);
        counter++;
        pthread_mutex_unlock(&mutex);
    }
    return NULL;
}

计算机网络

必修
学分: 4 大二下

学习网络分层模型、TCP/IP协议栈、HTTP协议等网络基础知识。

重点内容

  • OSI七层模型和TCP/IP四层模型
  • HTTP/HTTPS协议
  • TCP/UDP传输层协议
  • IP路由和DNS
  • Socket编程

代码示例

# Socket服务器
import socket

def start_server():
    server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    server_socket.bind(('localhost', 8080))
    server_socket.listen(5)

    print("服务器启动，等待连接...")
    while True:
        client_socket, addr = server_socket.accept()
        print(f"连接来自: {addr}")

        request = client_socket.recv(1024).decode()
        print(f"请求:\n{request}")

        response = """HTTP/1.1 200 OK
Content-Type: text/html


Hello, World!

"""
        client_socket.send(response.encode())
        client_socket.close()

# HTTP请求（使用requests）
import requests

response = requests.get('https://api.example.com/data')
print(f"状态码: {response.status_code}")
print(f"响应: {response.json()}")

避坑指南

🚫 不要忽视数学基础

高等数学、线性代数、离散数学这些课程看似没用，但它们决定了你的逻辑思维能力和后续学习算法的深度。

💡 建议：认真对待每一门数学课，它们会潜移默化地提升你的思维能力。

🚫 不要盲目追逐新技术

技术栈更新很快，但核心原理是不变的。不要盲目追逐每个新技术框架，要把基础打牢。

💡 建议：掌握基础原理后，学习新技术会非常快。先学计算机科学基础，再学具体技术。

🚫 不要只看不练

看教程和书籍固然重要，但如果不亲手写代码，永远学不会编程。

💡 建议：每学一个知识点，都要写代码实践。最好的学习方式就是做项目。

🚫 不要忽视英语

大部分优质技术文档、开源项目都是英文的。英语不好会严重限制你的技术发展。

💡 建议：养成阅读英文技术文档的习惯，逐步提升专业英语水平。

🚫 不要闭门造车

编程是一项社交活动。要学会使用GitHub、Stack Overflow、技术社区获取帮助和分享知识。

💡 建议：参与开源项目，加入技术社区，多和他人交流学习。

🚫 不要过度焦虑

看着别人的优秀履历会产生焦虑，但每个人的节奏不同。重要的是持续进步。

💡 建议：制定自己的学习计划，按自己的节奏稳步前进。

🚫 不要忽视身体健康

长时间编程会导致颈椎病、腰椎病、视力下降等问题。身体是革命的本钱。

💡 建议：每编程1小时休息10分钟，坚持运动，保持良好作息。

🚫 不要错过实习机会

实习经历对找工作非常重要。它能让你了解真实的工作环境，积累项目经验。

💡 建议：大二大三积极寻找实习机会，哪怕是小公司也能学到很多。
